## 🚀 Instalación y Uso

1. **Clonar o descargar el repositorio**
   ```bash
   git clone [repository-url]
   cd Sav_Solutions
   ```

2. **Abrir el proyecto**
   - Simplemente abre `index.html` en tu navegador
   - O usa un servidor local (recomendado):
     ```bash
     # Con Python
     python -m http.server 8000
     
     # Con Node.js (si tienes http-server instalado)
     npx http-server
     ```

3. **Personalización**
   - Edita los textos en `index.html`
   - Modifica estilos en `styles.css`
   - Ajusta funcionalidades en `scripts.js`
   - Reemplaza imágenes en `assets/`
